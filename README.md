# telecomx-analisis-datos
Análisis de datos TelecomX - Proyecto de Data Analytics por Aliana Trujillo
Este proyecto realiza un análisis de datos del caso TelecomX con el objetivo de identificar patrones en la cancelación de clientes (churn) y obtener insights que apoyen la toma de decisiones estratégicas.

El análisis incluye limpieza de datos, exploración de variables, visualización de información y generación de conclusiones sobre el comportamiento de los clientes.
